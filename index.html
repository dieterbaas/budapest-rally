⚙️ Setup / database-info

Items en vinkjes staan online in Supabase (tabel checklist), zodat alle vier dezelfde lijst zien — inclusief toegevoegde en verwijderde items.

De tabel heeft deze kolommen nodig (eenmalig in de SQL Editor draaien):

create table if not exists checklist (
  item_id text primary key,
  checked boolean default false,
  checked_by text,
  updated_at timestamptz default now()
);
alter table checklist add column if not exists category text;
alter table checklist add column if not exists label text;
alter table checklist add column if not exists note boolean default false;
alter table checklist add column if not exists sort double precision default 0;
alter table checklist enable row level security;
drop policy if exists "open access" on checklist;
create policy "open access" on checklist for all using (true) with check (true);
alter publication supabase_realtime add table checklist;
